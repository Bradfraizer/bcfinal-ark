Command Lines Explained

  • First, and most importantly, you must be in Singleplayer or have Admin priveledges for the server you are playing on in order for these codes to work.

  • Now, lets explain what you are looking at. "cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Weapons/PrimalItem_WeaponStonePick.PrimalItem_WeaponStonePick'" 1 0 0".

  • This is a command code to give your character a Stone Pick.

  • To most, that long command line probably looks like a foreign language, but it is actually the path of files that item must travel in order to appear in your in-game inventory.

  • For the most part, we don't need to concern ourselves with the intricate filing systems of our friends over at Studio Wildcard because the path that file travels is not always the same.

  • The most important information for you to familiarize yourself with is those numbers at the far back of the command lines.

  • These numbers, from left to right, decide how many of that item will appear in your inventory, the rarity of that item, or whether you want to receive a blueprint for that item.

  • For instance, if i wanted one Stone Pick in an advanced rarity, the end of my command line would look something like this "1 5 0" .

  • The "1" is stating that I would like 1 stone pick while the "5" tells me that i want higher than a primitive (0) rarity.

  • The rarity scale can be adjusted between 0-10 , 10 being Ascendent quality.

  • Furthermore, if i wanted that same Stone Pick, but as a blueprint I would use "1 5 1".

  • When it comes to that last number, it can only be adjusted between 0 and 1.

  • To summize, if you want just the item itself, leave that last slot at "0" and adjust to "1" if you're looking for just the blueprint.
